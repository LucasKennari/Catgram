**Possui sua própria API?**

>Para testar o projeto você tem 2 opções:
>
>**Opção 1:** Entrando no site do Deploy : [Site Catgram](https://catgram.onrender.com/)
>
>**Opção 2:** Fazendo um **Fork** do projeto na sua máquina.
